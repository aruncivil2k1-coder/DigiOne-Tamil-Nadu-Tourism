DigiOne – Tamil Nadu Tourism
===========================

UPLOAD INSTRUCTIONS:

1. Open your GitHub repository:
   DigiOne-Tamil-Nadu-Tourism

2. Delete all existing files in the repository.

3. Upload these files:
   - index.html
   - register.html
   - login.html
   - booking.html
   - reviews.html
   - Project PDF files (optional)

4. Go to Settings → Pages
   - Source: main branch
   - Folder: /root

5. Add your custom domain (GoDaddy).

6. Wait 5–10 minutes and reload site.

END
